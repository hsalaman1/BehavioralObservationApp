# Behavioral Observation App

A React application for documenting behavioral observation sessions with visit summaries and session notes.

## Features

- **Visit Summary**: Document the purpose of observation and synopsis of what took place
- **Session Information**: Track student name, observer, date, times, and location
- **Auto-save**: All data is automatically saved to browser localStorage
- **Clean UI**: Modern, responsive interface built with Tailwind CSS

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Usage

1. Navigate to the **Visit Summary** tab to enter:
   - Purpose of the observation
   - Synopsis of what took place during the session

2. Use the **Session Info** tab to record:
   - Student name
   - Observer name
   - Date and time
   - School/location

All data is automatically saved to your browser's local storage.
